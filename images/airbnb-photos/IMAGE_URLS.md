## ğŸ¨ Image Optimization (Recommended)

Before uploading to your website:

1. **Resize images** using TinyPNG.com or Squoosh.app:
   - Hero images: 1200x800px
   - Gallery images: 800x600px
   - Thumbnails: 400x300px

2. **Compress** to reduce file size:
   - Target: 150-300KB per image
   - Format: JPEG, quality 85%

3. **Rename** consistently:
   - `propertyname-purpose.jpg`
   - Example: `stone-hero.jpg`, `stone-bedroom1.jpg`
